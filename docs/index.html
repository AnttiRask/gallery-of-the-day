How It Works

1

Fetch Event

GPT-4o-mini retrieves a significant historical event for today's date.

2

Create Prompt

The event is transformed into an artistic image prompt.

3

Generate Image

DALL-E 3 creates a unique image based on the prompt.

4

Display

The image is stored and displayed in the Shiny app.
